# Triage Labels

The skills speak in terms of five canonical triage roles. This file maps those roles to the GitHub labels used in this repository.

| Canonical role    | GitHub label          | Meaning                                  |
| ----------------- | --------------------- | ---------------------------------------- |
| `needs-triage`    | `status:needs-review` | Maintainer needs to evaluate the issue   |
| `needs-info`      | `needs-info`          | Waiting on reporter for more information |
| `ready-for-agent` | `status:approved`     | Fully specified, ready for an AFK agent  |
| `ready-for-human` | `ready-for-human`     | Requires human implementation            |
| `wontfix`         | `wontfix`             | Will not be actioned                     |

When a skill mentions a canonical role, use its corresponding GitHub label from this table.
